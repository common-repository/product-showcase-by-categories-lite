Product Showcase By Categories is a WooCommerce plugin that allows you to display product showcase with selected category filters.

== Description ==

Product Showcase By Categories is a WooCommerce plugin that allows you to display product showcase with selected category filters. It displays products in a responsive carousel slider and uses ajax so users can navigate between categories without reloading the page. All you need to do is putting a simple shortcode and the plugin will do the rest.


**FEATURES (Lite Version)**

* Display products filtered by selected product categories
* Display featured products in a responsive carousel slider
* Use ajax when navigating between categories
* Shortcode to display product showcase
* Options to set layout style
* Select product template from plugin or from theme
* Fully templateable

**ADDITIONAL FEATURES (Pro Version)**

* Choose what products to be displayed:
  + Featured products
  + On-sale products
  + New products
  + Best selling products
  + Top rated products
* Option to show category images/thumbnails
* Option to show Add to Cart button

== Installation ==

There are 2 ways to install the plugin:
1. Go to plugin menu and upload the plugin zip file and then activate it. 
2. If you know how to use FTP, you can FTP all plugin files to `/wp-content/plugins/` directory.


For more information in using the plugin, please refer to documentation provided in the plugin file.

= How to use the plugin? =

Please read the documentation provided in the plugin file.

= I have installed the plugin but nothing shown, why? =

For the lite version, the plugin shows featured products, that means you need to set some products as featured on admin products listing.
